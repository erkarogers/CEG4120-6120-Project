\mbox{\hyperlink{epw__x_8py}{EPW\+\_\+\+X.\+py}} is a Jupyter Notebook that creates the input widgets for the EPW step of the simulation tool. The widgets are created using the ipywidgets library. The widgets are then displayed in a tabbed format using the App\+Layout widget. The widgets are then bound to the output widget. The output widget is then displayed in the App\+Layout widget. The App\+Layout widget is then displayed in the Jupyter Notebook.\hypertarget{_e_p_w__x_todo_epw}{}\doxysection{TODO}\label{_e_p_w__x_todo_epw}
